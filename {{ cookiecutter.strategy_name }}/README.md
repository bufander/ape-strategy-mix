## How to start

    pip install -r requirements-dev.txt
    
    yarn
    
    ape plugins install .

    ape plugins install . --upgrade
    
    ape compile

Create a `.env.sh` file following the `.env.example.sh` format and `source` it:
    
    source .env.sh

Run tests:

    ape test
